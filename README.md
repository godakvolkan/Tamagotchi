Ejderha evrimi ve oyunlaştırma mekanikleriyle desteklenmiş, modern bir sanal bebek deneyimi. Yumurtadan başlayın, ejderhanızı besleyin ve onu bir **Efsane Ejderha**'ya dönüştürün!

---

## 🚀 Projenin Amacı ve Özellikleri

Bu proje, geleneksel Tamagotchi mekaniklerini modern mobil oyun elementleriyle birleştirerek kullanıcıya uzun vadeli bir bağ kurma deneyimi sunar.

### 🎮 Oyunlaştırma Özellikleri (Gamification)
- **5 Aşamalı Evrim**: Ejderhanız seviye atladıkça fiziksel olarak değişir (Yumurta → Bebek → Küçük → Yetişkin → Efsane).
- **Mini Oyun "Düşen Et Yakala"**: Ejderhanızı beslemek için gereken coinleri aksiyon dolu bir mini oyunda toplayın.
- **Boss Savaşları**: Her 5 seviyede bir karşınıza çıkan "Boss Ejderha"yı yenerek büyük ödüller kazanın.
- **Rozet & Başarım Sistemi**: 10'dan fazla benzersiz rozeti (Hazine Biriktirici, Ejderha Katili vb.) toplayın.
- **Ekonomi & Market**: Kazandığınız coinlerle marketten özel iksirler, antidotlar ve XP kristalleri alın.
- **Görsel Şölen**: Dinamik parçacık efektleri (konfeti), nefes ateşi animasyonları ve haptik geri bildirimler.

## 🛠️ Kurulum ve Çalıştırma (Installation & Run)

### Ön Koşullar
- Node.js (v18+)
- Expo Go (Cihazda test için)
- Android Studio (APK derlemek için)

### Adımlar
1. **Depoyu Klonlayın**:
   ```bash
   git clone https://github.com/godakvolkan/Tamagotchi.git
   cd Tamagotchi
   ```
2. **Bağımlılıkları Yükleyin**:
   ```bash
   npm install
   ```
3. **Expo'yu Başlatın**:
   ```bash
   npx expo start
   ```
4. **Android'de Açın (Alternatif)**:
   - Android Studio üzerinden `/android` klasörünü açarak doğrudan derleyebilirsiniz.
4. **Android'de Açın**:
   - Expo Go uygulamasından QR kodu taratın.
   - Veya Android emulator için `a` tuşuna basın.
